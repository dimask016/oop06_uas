🧩 Mengapa menggunakan Enum?
• enum di Java digunakan untuk merepresentasikan sekumpulan konstanta yang tetap dan terbatas.
• Keuntungan: type-safe (tidak bisa menggunakan nilai di luar yang didefinisikan), mudah dibaca, dan dapat digunakan dalam switch statement.
• Alur status:
  1️⃣ WAITING_MANAGER → status awal setelah pengajuan dibuat.
  2️⃣ WAITING_DIVISION_HEAD → jika nominal > 50jt, request naik ke kepala divisi.
  3️⃣ WAITING_DIRECTOR → jika nominal > 250jt, request naik ke direktur.
  4️⃣ APPROVED → status akhir setelah disetujui sepenuhnya.
• Enum ini digunakan di seluruh sistem: di ProcurementRequest sebagai field status, di setiap Handler untuk pengecekan kondisi, dan di nextStatusAfter() untuk menentukan transisi.
