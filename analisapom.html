🧩 Apa itu pom.xml?
• pom.xml (Project Object Model) adalah file konfigurasi utama Maven yang mengelola build, dependensi, dan metadata project.
• <parent> → Menggunakan Spring Boot Starter Parent yang menyediakan manajemen versi default untuk dependensi Spring.
• <properties> → Mendefinisikan Java version 11 (sesuai dengan Spring Boot 2.7.0).
• <dependencies> → Menambahkan spring-boot-starter-web untuk membangun aplikasi web REST (embedded Tomcat, Spring MVC, Jackson).
• <build><plugins> → spring-boot-maven-plugin untuk mempackage aplikasi menjadi JAR yang bisa dijalankan (executable JAR) dan menjalankan perintah Maven Spring Boot.
• Dengan konfigurasi ini, aplikasi dapat dijalankan dengan mvn spring-boot:run atau java -jar target/procurement-approval-factory-1.0.jar.
⚙️ Maven | Spring Boot Starter Parent | Dependency spring-boot-starter-web | Plugin untuk executable JAR
