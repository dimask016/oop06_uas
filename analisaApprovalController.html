🍃 Catatan Spring Framework :
• @Controller → menandakan kelas sebagai controller Spring MVC, dapat mengembalikan view atau data (dengan @ResponseBody).
• @ResponseBody → membuat return value langsung ditulis ke HTTP response (bukan view resolver).
• @PathVariable → mengambil nilai dari template URL.
• ResponseEntity → kontrol penuh atas status HTTP, header, dan body response.
• Dependency Injection via konstruktor → field service bersifat final dan diinjeksi otomatis oleh Spring.
