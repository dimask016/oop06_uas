🧩 Apa yang dilakukan @SpringBootApplication?
• Anotasi @SpringBootApplication adalah gabungan dari tiga anotasi Spring:
  🔹 @Configuration – menandai kelas sebagai sumber definisi bean.
  🔹 @EnableAutoConfiguration – mengaktifkan auto-configuration Spring Boot (mengkonfigurasi bean berdasarkan dependensi yang ada di classpath).
  🔹 @ComponentScan – memindai package dan sub-package untuk menemukan komponen Spring (Controller, Service, Repository, dll).
• Method main adalah entry point Java standar. SpringApplication.run() akan:
  → Memulai embedded web server (Tomcat/Jetty/Undertow)
  → Membuat Spring IoC container
  → Melakukan auto-configuration dan component scanning
  → Menjalankan aplikasi hingga server dimatikan.
• Dengan struktur package com.example.procurementfactory, anotasi ini akan memindai semua sub-package (controller, service, handler, model) secara otomatis.
